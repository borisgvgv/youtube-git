# Comandos útiles de GIT

1. git init (inicializa el repositorio git en el ordenador)
2. git add . (añadir cambios)
3. git reset .
4. git commit -m
5. git checkout -- .
6. git log
7. git commit --amend
8. git checkout b rama-heroes

10. git branch -d rama-heroes
11. git push
